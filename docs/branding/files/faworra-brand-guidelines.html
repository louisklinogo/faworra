01 — Logo System

The mark

A continuous lowercase 'f' that flows into itself — signifying frictionless operation. The Forest Green dot is the living signal: the one element of color in a strict monochrome system. It appears wherever something is active, growing, or financially positive.

05 — Rules

Design rules

These are not guidelines — they are constraints. The aesthetic only holds when they are consistently applied.

  • Border radius Always 0px. No rounded corners anywhere in the product UI. Tags and badges may use 2px maximum.
  • Borders Always 1px solid — never 0.5px. The hard line is structural, not decorative.
  • Shadows None. Ever. Elevation is expressed through borders and background contrast only.
  • Color Forest Green (#0D3B2E) appears only on positive data, live states, and active navigation. Never as decoration.
  • Labels All metadata, field labels, timestamps, and identifiers use JetBrains Mono, uppercase, 10px, letter-spacing 0.10em.
  • Values All human-readable values — amounts, names, descriptions — use Inter, weight 400 or 500 only.
  • Hover states Inversion: Obsidian background → Ivory text. No color transitions. No easing longer than 150ms.
  • Logo Never stretch, recolor, or add effects to the mark. The green dot is always Forest (#0D3B2E) or Sandsong (#D4C5A8) on Forest backgrounds.
  • Logo spacing Minimum clear space equal to the height of the lowercase 'f' on all sides.
  • Currency Use ₵ (Cedi) as default. Always mono, always tabular figures for financial data.
